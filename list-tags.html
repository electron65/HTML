ol stands for ordered list. It is a list in order.

  1. Each item is in a list item.
  2. This is item number 2.
  3. This is item number 3, the last item is this order list.

ul stands for ordered list. It is a list in unorder. Which means there is no order.

  • Each item is in a list item.
  • This is an item.
  • This is an item, the last item is this unorder list.
